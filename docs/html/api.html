class ApplicationStatus(value)[source]¶

Choices for application status. These are set up as integers to facilitate the use of the status as hierarchical, i.e., APPROVED > CREATED.

Parameters

  • CREATED – Created in the system, but not otherwise touched.

  • PRE_REJECTED – Rejected before being sent out to reviewers for further evaluation.
